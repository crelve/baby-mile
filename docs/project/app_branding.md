# アプリブランディング — Baby Mile

## メイン名（グローバル）
- **英語名**: Baby Mile
- **正式表記（en）**: `Baby Mile — Milestones`
- **日本語表記**: `Baby Mile — 赤ちゃん成長記録`
- **決定理由**:
  - "Baby" は最も検索される育児カテゴリ最重要KW
  - "Mile" は milestone の短縮で覚えやすく、「歩み（mile）」の連想も持つ
  - 30字以内、シリーズ化しやすい（Baby Mile Pro / Baby Mile Family 等）
  - App Store内に直接競合なし（Baby Tracker / BabyConnect とは差別化）
