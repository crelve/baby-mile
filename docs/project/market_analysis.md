# 市場分析 — Baby Mile

> 作成: 2026-05-04 / Phase 1-3 統合
> 担当: 杏（CRO）

## 市場規模

- 全世界年間出生数: 約1.3億人
- 0-3歳人口: 世界数億人規模
- 日本の年間出生数: 約75万人（減少傾向だが、第一子親の課金意欲高）
- 米国の年間出生数: 約360万人

## 市場トレンド

- 親世代がSNSネイティブ化 → 「美しく残したい / シェアしたい」需要増
- スマホでの写真量爆発 → 「整理できない問題」が顕在化
- 育児可視化アプリは2010年代から定着、後発のビジュアル特化に需要
- 月齢別発達情報は信頼できるソースが少ない → 公的データを束ねた価値

## カテゴリ動向

- **Lifestyle / Parenting** が主力。Medical も二次選択肢
- 米国TopGrossing（Parenting）には Glow Baby / Ovia / BabyCenter
- 日本は「ぴよログ」が強いが、UI 古めで写真特化の余地あり

## 結論

需要・市場・課金力すべて◎。マイルストーン特化のサブカテゴリで参入余地大。
