# 機能仕様 — Baby Mile

> 作成: 2026-05-04 / Phase 6-7
> 担当: 今田 美桜 + 本田 圭佑

## 1. オンボーディング

| ステップ | 画面 | 入力 | 出力 |
|---------|------|------|------|
| 1 | 子供の名前 | name | Childオブジェクト初期化 |
| 2 | 生年月日 | DateTime | 月齢自動算出 |
| 3 | 1件目記録誘導 | maybe photo + milestone | 初回マイルストーン |

`aha`：1件目のマイルストーンを記録し、カードを生成して見せる。

## 2. ホーム / ダッシュボード

- アクティブな子供の月齢サマリー
- 最近のマイルストーン3件
- 「今月チェックすべき発達項目」プレビュー
- CTA: 「マイルストーンを記録」「カードを作る」

## 3. マイルストーン記録

- プリセット選択 or 自由入力
- 写真選択（カメラ / ライブラリ）
- 日付選択（既定は今日）
- メモ（200字以内）
- 保存 → カード生成画面へ自動遷移（任意）

## 4. 月齢別チェックリスト

- 月齢タブ（0, 1, 2, ..., 36）
- カテゴリ別（運動 / 言語 / 社会 / 認知）
- 達成チェック → そのままマイルストーン記録に遷移

## 5. マイルストーンカード生成

- レイアウトテンプレ（無料3 / Premium全部）
- フォトフィルター（無料2 / Premium全部）
- 文字色 / フォント切替
- 書き出し: SNS最適サイズで保存・共有

## 6. タイムライン

- 全マイルストーンを時系列降順
- 月齢ごとに区切り表示
- タップで詳細 / カード再生成

## 7. 設定

- 子供の追加 / 編集 / 削除
- 通知ON/OFF・頻度
- 言語切替（端末準拠 + 手動）
- プレミアム購入 / 復元
- 法令（プライバシー / 利用規約）
- アカウント（Apple Sign-In）

## 8. ペイウォール

- 表示タイミング: 初回3件記録後 / 4K書き出し / 無料制限到達時
- プラン: 月額 ¥480 / 年額 ¥6,000 / 買い切り ¥4,980
