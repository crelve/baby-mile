# アプリコンセプト — Baby Mile

## エレベーターピッチ

> **「はじめて笑った日、忘れない。」**
> 0-3歳の赤ちゃんの "はじめて" を、写真+日付+メモで美しく残す成長記録アプリ。
> 月齢別の発達チェックリストとSNS映えするマイルストーンカード自動生成で、
> 育児の感動的な瞬間を最高の形でストックする。

---

## なぜ作るか（WHY）

- 子供の成長記録は **最も感情的動機が強い領域**。継続率・LTVが高い
- 既存の育児記録アプリ（ぴよログ / Baby Tracker / Tinybeans）は授乳・睡眠・写真共有が中心で、**マイルストーン特化が弱い**
- "はじめて" は人生で一度しか起きない瞬間。雑な ToDo リストではなく **作品として残したい** ニーズに応える
- マイルストーンカードのSNSシェアで自然な口コミ拡散を設計する

## 何を作るか（WHAT）

| カテゴリ | 内容 |
|---------|------|
| マイルストーン記録 | 50+のプリセット（笑った/寝返り/はいはい/つかまり立ち/初語/初歩 等）+ 自由追加 |
| 月齢別チェックリスト | 0-3歳を月齢で区切り、WHO/AAPの発達目安を一覧化 + 達成チェック |
| マイルストーンカード | 写真+日付+月齢+ひとこと を自動レイアウト。Instagram/LINE/X 向けサイズ |
| タイムライン | 全マイルストーンを時系列スクロールで振り返る |
| 家族共有 | パートナー・祖父母にマイルストーンを共有（Premium） |

## 誰のためか（WHO）

- **プライマリ**: 0-3歳の子供を持つ25-40代の親（特に第一子）。「初めて」を美しく残したい層
- **セカンダリ**: 祖父母（孫の成長確認）、保育士、出産祝いで勧める友人

## 差別化（HOW）

- マイルストーン**特化**：機能を絞ってチェックリストとカード生成に集中
- **美しいビジュアル**：石原さとみ（CCO）と組んだフォト中心UI
- **月齢別目安DB**：WHO/AAPベースの公的指標を内蔵
- **SNS流用前提**：縦長/正方形/横長の3サイズを1タップ切替
