# 競合分析 — Baby Mile

> 作成: 2026-05-04 / Phase 3
> 担当: 杏（CRO）+ 今田 美桜（CPO）

## 直接競合

| アプリ | 強み | 弱み | 差別化ポイント |
|-------|-----|-----|--------------|
| **ぴよログ** | 国内シェアNo.1、授乳・睡眠の高速入力 | UI地味、マイルストーンは付属機能、SNS不向き | マイルストーン特化＋ビジュアル＋SNSカード |
| **Baby Tracker (Nighp)** | グローバル展開、授乳トラッカー第一想起 | マイルストーン機能薄い、UIが古い | 美しいカード生成＋月齢別チェックリスト |
| **Tinybeans** | 写真共有の家族SNS、米国で強い | "はじめて"の構造化なし、流れていく | "はじめて"を作品化して残す。写真共有の上位互換 |
| **Wonder Weeks** | メンタルリープ理論で根強いファン | 記録機能なし、読み物中心 | 発達目安に加えて記録＋シェア機能 |
| **Glow Baby** | 米国で強い、機能豊富 | 機能過多で初心者には複雑 | "はじめて"だけに絞ったシンプルさ |

1. **マイルストーン特化** — 既存育児アプリの「おまけ機能」を本体機能化

3. **発達目安DB** — WHO/AAPベースで信頼性付与
4. **家族共有** — Premium動線として
5. **ロングテールKW** — 「マイルストーン × 赤ちゃん」で空白攻略
