# 画面構成 — Baby Mile

> 作成: 2026-05-04 / Phase 7

## 画面一覧

| ID | 画面 | 主要要素 |
|----|------|---------|
| S01 | Onboarding | 名前 / 生年月日 / 1件目記録 |
| S02 | Home | サマリー + 最近の記録 + CTA |
| S03 | Record | プリセット選択 / 写真 / 日付 / メモ |
| S04 | Checklist | 月齢タブ + カテゴリ別チェックリスト |
| S05 | Timeline | 時系列リスト + フィルタ |
| S06 | Card Generator | テンプレ + プレビュー + 共有 |
| S07 | Settings | 子供管理 / 通知 / プラン / 法令 |
| S08 | Paywall | プラン3つ + トライアル CTA |
| S09 | Detail | マイルストーン詳細 |

## ナビゲーション構造

- ボトムナビ4タブ: Home / Checklist / Timeline / Settings
- フローティング+ボタン: Record 起動
- Card Generator は Record 完了 or Detail から開く
